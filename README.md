# Chop Chop

## To get going with it here are the commands

1. create directory in Sites folder
2. create public_html folder
3. git clone git@gitlab.iweb.co.uk:labs/chopchop.git
4. npm install
5. gulp

## Working on the project

1. CD to directory
2. gulp watch
